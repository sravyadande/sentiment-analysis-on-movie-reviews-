# sentiment-analysis-on-movie-reviews

 Project Overview

This project performs Sentiment Analysis on movie reviews using Natural Language Processing (NLP) and Machine Learning. The model classifies reviews as either Positive or Negative based on the text provided by users.

The project demonstrates the complete machine learning workflow, including data preprocessing, feature extraction, model training, evaluation, and prediction.

---

 Features

- Text preprocessing (lowercasing, punctuation removal, stopword removal, stemming/lemmatization)
- Feature extraction using TF-IDF
- Machine learning model for sentiment classification
- Predicts sentiment for new movie reviews
- Model evaluation using accuracy, precision, recall, and F1-score

 Installation

1. Clone the repository

git clone https://github.com/sravyadande/Sentiment-Analysis-on-Movie-Reviews.git

2. Navigate to the project directory

cd Sentiment-Analysis-Movie-Reviews

3. Install the required libraries

pip install -r requirements.txt

---

 Usage

Run the Jupyter Notebook or Python script:

jupyter notebook

or

python sentiment_analysis.py

Example:

Input

The movie was fantastic with an amazing storyline.

Output

Positive
